(static) getRequest(url, contentType, callback)

Builds an XMLHttpRequest according to [MDN] https://developer.mozilla.org/en-US/docs/Web/API/XMLHttpRequest
Parameters:
Name Type Description
url string The URL to send the request to
contentType string Adds a request header with this content type, may be null or empty ('')
callback function A function to be called when the request has returned a response. In case of any HTTP status other than "200 OK", this MAY return null!
